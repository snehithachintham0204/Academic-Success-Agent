# 🎓 Academic Success Coaching Agent

The **Academic Success Coaching Agent** is an AI-powered web application designed to help students improve their academic performance through personalized study planning and intelligent recommendations.

This project was developed using **Vercel v0 (vibe coding)** to rapidly build an interactive interface that allows students to manage subjects, track assessments, and generate adaptive study schedules.

---

## 🚀 Project Overview

Students often struggle with managing multiple subjects, assignments, and exams.  
The Academic Success Coaching Agent analyzes academic inputs such as **subjects, scores, and exam deadlines** to automatically generate a **personalized weekly study schedule**.

The system highlights weak subjects and suggests targeted study focus areas to help students improve learning outcomes.

---

## ✨ Key Features

- 📊 **Performance Tracking**  
  Allows students to enter subjects, exam dates, and assessment scores.

- 📅 **Automated Study Schedule Generation**  
  Generates a weekly study plan based on subject importance and performance levels.

- ⚡ **Smart Recommendations**  
  Identifies weaker subjects and suggests focused study sessions.

- 🌐 **Interactive Web Interface**  
  Built using **Vercel v0** for a responsive and user-friendly experience.

---

## 🧠 How It Works

1. Students input their subjects, exam dates, and assessment scores.
2. The system analyzes performance across different subjects.
3. Weak subjects are identified based on score trends.
4. A personalized weekly study schedule is generated.
5. Students receive suggestions to focus on high-priority topics.
